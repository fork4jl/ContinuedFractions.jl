This module provides functionality for working with continued fractions.

ContinuedFractions.ContinuedFraction — Type
ContinuedFraction{T<:Integer}

An abstract type representing a continued fraction with elements of type T, where T is a subtype of Integer. This type serves as a base for defining specific types of continued fractions.
